# MULTI 111 Syllabus Reference (Prerequisite)

Source: Course_Syllabus_Adobe_Photoshop_I.pdf

This is the prerequisite course for MULTI 211. Use this to understand what
students should already know coming into MULTI 211 — do not re-teach these
topics unless reviewing is pedagogically necessary.

## Course Info
- Instructor: Mike Walter
- College: Crafton Hills College
- Prerequisites: None
- Credits: 3
- Delivery: Fully online, weekly recorded lectures

## Student Learning Outcomes (MULTI 111)
- Define and work with raster-based artwork for screen and print
- Correct and enhance existing photographs
- Create original artwork and compositions

## Topics Covered (Weekly Sequence)
1. Interface, navigation, preferences, digital image ethics
2. Image size, resolution, color modes, cropping, saving
3. Layers, alignment, transforms, smart objects
4. Selections, layer masks, clipping masks
5. Luminance/color correction, adjustment layers, blend modes
6. Retouching, blurring/sharpening, liquify, generative AI
7. Vector tools, shape tools, pen tool
8. Text creation/formatting, fonts, text on paths
9. Filters, smart objects, layer effects, gradients, textures
10. Painting tools, brush engine, custom brushes
11. Thanksgiving recess
12. Rulers, guides, grids, print/web publication
13. Work week
14. Final project

## Grading
- Standard letter grade (A–F)
- Discussions: 10 pts each
- Quizzes: 20 pts each
- Class Projects: 20 pts each
- Homework Projects: 30 pts each
- Final Project: 90 pts
- Late work: 10% deduction (except quizzes)
