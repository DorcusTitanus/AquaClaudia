# MULTI 211 Project Rules & Standards

Source: MULTI_211_Project_Rules.pdf (Authoritative Project Reference)

## Course Scope
- 13-week asynchronous online course
- Prerequisite: MULTI 111
- Primary tool: Adobe Photoshop
- Focus on non-destructive professional workflows

## Pedagogical Framework
- Hidden Horton Absorb–Do–Connect model
- 6 instructional levels plus final portfolio week
- Emphasis on revision, collaboration, and authentic practice

## Gamification & Points
- Each level totals 40 points standard (some levels with extra steps: 50 pts)
- Tutorial: 5 points (File Submission 3 + Tutorial Steps Completed 2)
- Checkpoint quiz: 5 points (min_score 4.0 to pass)
- Project / Peer Review: 25 points (5 criteria × 5 pts binary, hide_points)
- Weekly Journal: 5 points (Reflection Depth 3 + Completeness 2)
- All quizzes titled "Checkpoint"

## Project Rules
- All major projects are group-based
- Flexible roles supported for UDL alignment
- Each project must include purpose, steps, deliverables, and evaluation criteria

## Canvas Structure
- Modules follow: Overview, Absorb, Do, Connect, Assessment
- Semantic headings only (H2–H4)
- Page title and first H2 may not repeat

## Accessibility & Compliance
- WCAG 2.1 Level AA compliance required
- All images include alt text under 120 characters
- Videos must be captioned; audio requires transcripts
- Instructions may not rely on visuals alone

## External Links & Tools
- All external links open in new tabs
- Links must include Canvas `inline_disabled` class and color override
- Third-party tools must be accessible or optional

## File & Technical Standards
- Photoshop files are screen-based at 72 ppi
- Maximum canvas size: 1920 × 1280 pixels
- Designed for manageable upload and review

## Assessment Philosophy
- Assessment prioritizes understanding over speed
- Peer review uses structured prompts or rubrics
- Reflection activities are guided

## Badges & Visual Standards
- Locked badge: gray question mark embroidery style
- Earned badge Level 1: Mask Architect
- File naming and dimensions are locked standards

## AI Use
- AI tools may support learning but not replace decision-making
- Students must demonstrate understanding and intent
