# Feedback: Jeff vs. JEFE vs. Jefe

**Date:** 2026-04-16

## The rule

- **Jeff Wilson** — formal name. Use in any public-facing context
  (websites, documents, signatures, Canvas attributions, footer credits,
  email signatures).
- **JEFE** (all caps) — self-styling, rare, only where he has chosen to
  present himself that way.
- **Jefe** (lowercase J, no all-caps) — what close friends and family
  call him. He does not refer to himself this way and does not want the
  public to see him referenced this way.
- **Jefe Wilson** — never. Wrong combination. Don't use.

## Where I got this wrong

Put "Made by Jefe Wilson" in the footer of wilsonidt.com/cards/.
Jefe corrected directly: "Name is JEFF not JEFE, that's for friends only
and if we're doing semantics, JEFE only never Jefe Wilson. Jeff Wilson or
Jefe and Jefe only with friends—they call me that, I don't call myself
that."

## How to apply

- **Anything shown publicly on Jefe's websites, Canvas content, course
  materials, print output, etc.** → "Jeff Wilson"
- **Internal memory files, informal notes, my own references to him
  as the user/client in this project** → "Jefe" is fine (that's how the
  project refers to him throughout)
- **Never use "Jefe Wilson"** as a combined form anywhere
